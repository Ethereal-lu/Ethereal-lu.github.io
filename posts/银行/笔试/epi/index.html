1、入门#

1、时间型问题#

题型特征：题干只有时间，没有效率、效率比

解题步骤：

  1. 设工作总量 W 为题干所给时间的最小公倍数
  2. 用公倍数 W 和所给时间 t 算出效率 P
  3. 用 W 的所需的 P 算出结果

2、平均速度#

定义： 平均速度 = 总路程 / 总时间

等距离平均公式：v = 2 * v1 * v2 / (v1 + v2)

3、时针分针重合规律#

  • 一天中第一次重合一定是 0 点 0 分
  • 每个小时内最多重合一次
  • 公式：时针分针在 n 时重合的时刻为：n 时 60/11 * n 分(0 <= n <= 11)
    • 将 11 代入公式可知，11 时重合时刻为 0 时 0 分，故一天 24 小时内，时针分针重合次数为 22 次。
    • 当题目为 24 小时制时将其化为 12 小时，即让求 18 时的重合时刻，将 6 带入公式，即 18 时 60/11 * 6 分
    • 除了 0 和 11，其余数代入公式都不可能整除，故正确选项的分母一定是 11，可直接排除分母为其他数字的选项

4、三元容斥问题#

题型特征：喜欢 A 的有几人，喜欢 B 的有几人，喜欢 C 的有几人，都喜欢的有几人。。。。

计算公式：P = A + B + C - AB - AC - BC + ABC + (x)。公式中的 x 是那些都不喜欢的人

6、鸡兔同笼#

例题：有 35 个头，94 只脚，问鸡兔各几只？

  1. 假设都是鸡，则应有 70 只脚
  2. 在 1 中每将一只兔子当成鸡，就少算 2 只脚，共少算 24 只脚，即将 12 只兔子当成了鸡
  3. 共 12 只兔子，23 只鸡

7、切割问题#

  • 直线分割平面：平面数 = n * (n + 1) / 2 + 1，n 为直线数
  • 直径和弦分割圆：n 条直径和一条弦可将圆分割为 3n + 1 部分

8、空瓶换水#

n 个空瓶换 m 个水，即 (n - m) 个空瓶可换 m 个水。有 N 个空瓶，则可换 N / (n - m) * m 个水

9、排队最短时间#

例题：A、B、C、D、E打水各需要 1、2、3、4、5分钟，问排队加等待的时间最短是多久？

按所需时间从小到大排，然后最大的乘1，次大乘2。。。，最后相加就是最短时间。这道题为 5A+4B+3C+2D+1E

10、煎蛋问题#

例题：一个蛋有两个面，一个锅可同时煎两个蛋，一面煎一分钟，问 23 个蛋最快需要煎多久？

公式：一个蛋的面数 / 一个锅可同时煎的面数 * 每一面需要的时间

11、割绳子问题#

1根绳子，对折 n 次后，剪 m 刀，会变成几段？

公式：绳子的段数 = m * 2 ^ n + 1

解释：绳子不对折，剪 m 刀成 m + 1 段；对折 n 次后，对折处有 2 ^ n 根绳子，剪一刀相当于剪 2 ^ n 刀，故剪 m 刀，相当于剪 m * 2 ^ n 刀，成 m * 2 ^ n + 1段。

12、牛吃草问题#

例题：牧场的草每天均速生长，10头牛20天吃完，15头牛10天吃完，问25头牛多久吃完？

公式：M = (牛1 - x) * t1 = (牛2 - x) * t2 = (牛3 - x) * t3

  • M：草场原有草量
  • 牛n：牛的头数
  • tn：吃草的时间
  • x：草生长的速度

13、换元法#

例题：(1 + 0.19 + 0.88) * (0.19 + 0.88 + 0.94) - (1 + 0.19 + 0.88 + 0.94) * (0.19 + 0.88) = ?

观察式子看到 0.19 + 0.88在每个项中都有，可以将它看作整体用 x 替换，可得：(1 + x) * (x + 0.94) - (1 + x + 0.94) * (x) = ?

展开销项后得到结果为 0.94。

对于这种题，可以将 1 看作头，0.94 看作尾，则式子中各项为：有头没尾 ✖ 有尾没头 ➖ 有头有尾 ✖没头没尾

这样的式子，结果直接为 头 ✖ 尾

14、流水行船问题#

顺水速度 = 船速 + 水速；逆水速度 = 船速 - 水速。这辆不常考

船速 = (顺水速度 + 逆水速度) / 2

水速 = (顺水速度 - 逆水速度) / 2

16、日历问题#

例题：2010年1月1日是星期五，问2012年1月1日是星期几？

平年：365天 -> 52周零 1 天

闰年：366天 -> 52周零 2 天

整周不影响计算星期几，重要的是零几天。该题 2010 和 2011 年都是平年，多出两天，故答案为 星期日
