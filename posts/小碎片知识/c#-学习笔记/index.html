1、基础概念#

1.1、.Net 概念#

通常所说的 .Net 包括 .Net 平台和 .Net FrameWork 框架。它们两者的关系即 jvm 和 jre 的关系。

.Net FrameWork 框架由两部分组成，分别是 .Net 类库和 CLR（公共语言运行时）。

1.2、C# 语言#

C# 是一种编程语言，可以开发基于 .Net 平台 的应用。

1.3、.Net 作用#

  • Winform 开发桌面应用程序
  • ASP.NET 开发 web 网站
  • Unity3D 开发游戏和虚拟现实

1.4、.Net 两种交互模式#

  • C/S：客户机（Client）/服务器模式。客户端需要安装专用的客户端软件，如 微信。
  • B/S：浏览器（Browser）/服务器模式。客户端只需要安装浏览器即可。

2、语法规则#

C# 的基础语法规则几乎与 java 一样。

2.1、基本类型#

与 java 一样，包括 byte、short、int、long、float、double、bool等，u开头表示无符号。

如果一个小数是浮点类型，则其后需加个 f ，否则默认为 double 型。

2.2、字符串#

类名为 string ，小写。它的使用和 java 几乎一样，注意方法名为大写开头。

字符串格式化：string str = $"{name}的年龄为{age}"，其中name和age为之前声明的变量。

2.3、函数#

C# 的函数名要大写开头。

返回值类型 函数名 （参数列表）{任意指令；return 返回值；}

// 示例
float MulAdd(float val1, float val2, float val3) {
    return val1 + val2 + val3;
}

2.4、switch 和 enum#

// 对枚举进行switch，一种书写思想，不针对与特定语言。
static void SwitchEnum(GenderType genderType) {

    switch (genderType) {
        case GenderType.Male: Console.WriteLine("男"); break;
        case GenderType.Female: Console.WriteLine("女"); break;
        case GenderType.Other: Console.WriteLine("其他"); break;
    }
}

enum GenderType {
    Male,
    Female,
    Other
}

2.5、ref 关键字#

ref 可以让变量在函数内被修改，可以让函数有多个返回值。即引用，传地址。

int TestRef(int val1, ref int val2) {
    val2 -= 1;
    return val1 + 1;
}
int val1 = 0; int  val2 = 3;
int val3 = TestRef(val1, ref val2);
// val3 == 1   val2 == 2

3、面向对象#

3.1、类#

类修饰符：public、internal、 partial、abstract、sealed、static、new、protected、private、protected internal

成员修饰符：public、protected、private、internal、protected internal、abstract、virtual、override、readonly、const、sealed、new

partial：部分类，可以将一个类分成几部分写在不同文件中，最终编译时将合并成一个文件，且各个部分不能分散在不同程序集中

类内成员如果没有修饰符，默认为 private。

partial class Person {
    private const int MaxAge = 150;
    public static bool canFly = false;
    int age;               // 相当于 private int age;
    public string? name;   // string? 可以用 null 初始化
    
    public Person(int age, string name) {   //  显示声明有参构造后，就不再赠送无参构造。和 java 一样。
        this.age = age;
        this.name = name;
    }

    public void SetAge(int age) {
        this.age = age;
    }

    public static void SetAge(Person person, int age) {
        person.age = age;
    }

    public static void PrintTypeName() {
        Console.WriteLine("TypeName = Person");
    }
}

// 与上方是同一个类，写在不同的文件中
partial class Person {
    public int Age {
        get {
            Console.WriteLine($"getAge{age}");
            return age;
        }
        set {
            Console.WriteLine($"setAge{age}=>{value}");
            age = value;
        }
    }

    public static bool CanFly {   // 与 Age 的作用一样，是上方的简化写法。
        get => canFly;
        set => canFly = value;
    }
}

属性 Property ：如上方的 Age、Canfly，看起来像变量，其实是函数。可进行下方操作：

Person person = new Person(18, "qin");
person.Age = 20;            // 调用 Age 中的 set 方法，value 的值即为 20；
int myAge = person.Age;     // 调用 Age 中的 get 方法。
Person.Canfly = true;       // 调用 CanFly 中的 set 方法。
bool isCanFly = Person.Canfly;   // 调用 CanFly 中的 get 方法。
